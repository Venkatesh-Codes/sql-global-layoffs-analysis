# 📊 Global Layoffs Data Analysis Using SQL

## Overview

This project demonstrates the use of SQL for data cleaning and exploratory data analysis (EDA) on a real-world global layoffs dataset. The objective was to transform raw data into a clean and structured format and uncover meaningful insights regarding workforce reductions across companies, industries, countries, and time periods.

Through this analysis, SQL was used not only as a querying language but also as a powerful tool for data transformation, trend identification, and business intelligence generation.

---

## Business Problem

Layoffs have become a significant indicator of economic and industry-wide shifts. Understanding where, when, and how layoffs occur can provide valuable insights into market conditions, industry performance, and organizational challenges.

This project aims to answer questions such as:

* Which companies experienced the highest layoffs?
* Which industries were most affected?
* Which countries recorded the highest workforce reductions?
* How have layoffs changed over time?
* What trends can be observed across different years and months?

---

## Dataset

The dataset contains information on layoffs reported across various companies worldwide, including:

* Company Name
* Industry
* Country
* Total Employees Laid Off
* Percentage Laid Off
* Funding Raised
* Date of Layoff
* Company Stage

---

## Project Workflow

### 1️⃣ Data Cleaning

The raw dataset was cleaned and standardized using SQL.

Key cleaning steps included:

* Removing duplicate records
* Standardizing text values
* Formatting date columns
* Handling null and missing values
* Correcting inconsistent industry and company information
* Improving overall data quality for analysis

---

### 2️⃣ Exploratory Data Analysis (EDA)

Performed comprehensive analysis to identify patterns and trends in layoffs.

Areas analyzed:

* Total layoffs by company
* Total layoffs by industry
* Total layoffs by country
* Year-over-year layoff trends
* Monthly layoff trends
* Companies with 100% workforce reductions
* Ranking companies by layoffs using Window Functions
* Rolling totals and cumulative trends

* Determined industries most impacted by layoffs.
